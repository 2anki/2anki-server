# Notion 2 Anki

## Install dependencies

```
npm install
```

### Run in development mode

```
npm run dev
npm run start
```

### Build package (OSX)

```
npm run build
npm run package-osx
```

### Build package (Linux)

```
npm run build
npm run package-linux
```

### Build package (Windows)

```
npm run build
npm run package-windows
```
